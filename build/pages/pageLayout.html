Spacing

Spacing utilities allow you to insert generic spacing to elements and components.

The classes take the following form .uTypeDirectionsize

  • A   All
  • T   Top
  • R   Right
  • B   Bottom
  • L   Left
  • H   Horizontal
  • V   Vertical
  • n   None
  • xxs   5px
  • xs   10px
  • s   15px
  • m   20px
  • l   25px
  • xl   35px
Example:
This box uses the helpers and as a result has a large margin at the bottom and large padding on all sides.
This box has large padding on the left. 
<div class="pink box uMarginBl uPaddingAl">
  This box uses the helpers and as a result has a <u>large margin at the bottom</u> and <u>large padding on all sides</u>.
</div>

<div class="blue box uPaddingLl">
  This box has large padding on the left.
</div>

Page Layout

Synergetic makes use of the Bootstrap Grid for basic page layout.

As a default and for simplicity use .col-xs- as this targets all screen sizes. Additional classes, e.g .col-lg- s, .col-sm- can be added for layouts which need to respond to different screen widths.
