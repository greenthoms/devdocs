Visual Design

Typeface

The user interface font is Roboto. The typeface is available open source under an Apache License.

Font Sizes

The default font size in Synweb is 14px. This is set on the <body> element and does not need to be added to individual elements or modules.

Pixel sizes Description
12px Small text <small> can be used help text and as secondary supporting text.
14px The default for all type is 14px.
16px Use the class .large for text which needs to be bigger than the default.

18px

 H4 Headings are set at 18px.

21px

 H3 Headings are set at 21px.

28px

 H2 Headings are set at 28px.

35px

 H1 Headings are set at 35px.

Font Styles

Element Description
Regular The default font style all type is Regular.
Bold Use the <strong> to signify important text. Use .bold for bold styling text on text without special importance.
Italic text Use the <em> to add emphasis to a word or phrase.
Small text Small <small> can be used help text and as secondary supporting text.
Lead Use the class .lead is used to introduce a section.

Font Weights

The following font weights can be used where appropriate:

  • Light 300
  • Normal 400
  • Normal 400 Italic
  • Medium 500
  • Bold 600

Line lengths and readability

Where possible aim for a maximum line length of 100 characters per line (including spaces).

Text Colours

This is main default colour for all text. #333333

Grey text can be used for secondary text. #757575

Blue text is used for links. #0275D8
