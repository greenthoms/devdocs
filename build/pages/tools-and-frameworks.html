Bootstrap

Bootstrap v3.4.4 is used as the foundation for all UI new developments in Synweb v67.

To avoid conflicts with legacy CSS, the prefix .syn is used to increase the specificity of all Bootstrap styles—the prefix has been included in the bootstrap/bootstrap.less file so styles can be written normally—the prefix is added automatically when the Less is compiled.

Kendo

Kendo UI is our primary library for user controls for all new UI developments in Synweb v67.

These user controls superseed DevExpress but will be used alongside DevExpress controls until a full transition is complete. Read the Kendo UI documentation and try the demos for an introduction to the user controls. Specific examples are shown in the Forms and Components sections of this document.

Less

Less is our pre-processor of choice as it is used and maintained in both Kendo and Bootstrap.

Less is a CSS pre-processor, meaning that it extends the CSS language, adding features that allow variables, mixins, functions and many other techniques that allow you to make CSS that is more maintainable, themable and extendable.

Less folder structure

The folder and file structure for is based on Harry Roberts ITCSS

CSS/LESS/
|
|-- base/                     # Default files
|   |-- _variables.less       # Variables used in other less files.
|   |-- _base.less            # Global styles, usually applied to the <body> or <html>.
|   ...
|
|-- libraries/                # Bootstrap, Kendo UI and other frame works or plugins.
|   |-- _animate.css          # CSS animation library
|   |-- kendoCustom/          # All overrides to the Kendo framework
|   `-- bootstrap/            # All the bootstrap less files
|       |
|       `-- bootstrapCustom/  # All global overrides to the Bootstrap framework
|
|-- modules/                  # Custom modular CSS created by Synergetic.
|   |-- _purchaseReq.less     # imports for all mixins + global project variables
|   |-- _userMessages.less    # buttons
|   |-- _welcomeWidgets.less  # Welcome page widgets
|   ...
|
|-- utilities/                # Helpers and overrides.
|   |-- _type.less            # 
|   |-- _utilities.less       # UI Helpers. Use only if a helper does not exist in Bootstrap.
|   ...
|
|-- app.less                # Primary Less file
`-- app.css          	    # Output CSS. The compiled file loaded by the browser.
